# Product Card — E-commerce UI 🛍️

> A modern e-commerce product card with color selection, pricing display, and smooth interactions.

![Project Preview](project5-converse.png)

## 🔗 Live Demo
[View Project](https://mellifluous-jelly-d3824e.netlify.app/)

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript

## ✨ Features
- Dynamic color selection
- Pricing display
- Add to cart button
- Product details section
- Clean and modern UI
- Fully responsive design
- Smooth hover interactions
